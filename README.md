# plant-disease-detector
AI based plant disease detection system using Python and TensorFlow
# Plant Disease Detection System

# Project Overview
An AI-based system that detects plant diseases from leaf images using deep learning.

# Technologies Used
- Python
- TensorFlow
- Flask
- HTML/CSS
- OpenCV

# Features
- Upload plant leaf image
- AI predicts disease
- Shows treatment information
